📱 Projet Flutter PsyAI prêt pour test mobile (ou web)

✅ Étapes à suivre :

1. Ouvrir dans VS Code :
   $ code psy_ai_mobile_ready

2. Installer les dépendances :
   $ flutter pub get

3. Tester en local (web) :
   $ flutter run -d chrome

4. Compiler et tester sur téléphone :
   $ flutter build apk
   Puis installer sur ton téléphone Android via l’APK

5. Déployer sur Firebase :
   $ flutter build web
   $ firebase deploy

🎯 Assure-toi d’avoir Firebase CLI et flutter bien configurés.
